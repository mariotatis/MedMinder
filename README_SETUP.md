# Project Setup

To generate the Xcode project (`.xcodeproj`), this project uses **XcodeGen**.

## Prerequisites
- **Homebrew** (if not installed)
- **XcodeGen**

## Installation
1. Install XcodeGen:
   ```bash
   brew install xcodegen
   ```

2. Generate the project:
   ```bash
   xcodegen generate
   ```

3. Open the project:
   ```bash
   open MedMinder.xcodeproj
   ```
